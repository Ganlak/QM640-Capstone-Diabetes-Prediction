## Folder Structure
- `/data` - Dataset files and documentation
- `/notebooks` - Jupyter notebooks (EDA, modeling, evaluation)
- `/src` - Python source code
- `/models` - Trained model artifacts
- `/reports` - Synopsis and final report
- `/figures` - Plots and visualizations

## Dataset
- **Source:** [CDC BRFSS 2015 Diabetes Health Indicators](https://www.kaggle.com/datasets/alexteboul/diabetes-health-indicators-dataset)
- **Size:** 253,680 observations × 22 variables
